4. Permissions & How We Use Them

Zloop requires the following Android permissions to operate. These permissions are used strictly as described below. We do not use them for any other purpose.

Display Over Other Apps

Draws the Zloop animated eye on top of all apps. Used only to render the visual overlay.

Accessibility Service

Detects which app is currently active to enable contextual reactions (e.g., reacting to food apps, camera, social media). Does not read messages, passwords, or personal content.

Microphone

Listens for the wake word "Hey Zloop" and processes voice commands using Android's on-device SpeechRecognizer. Audio is never stored or recorded by Zloop.

Phone State

Detects call start and end events so Zloop can pause its microphone during calls and react when calls finish. No call content or numbers are accessed.

Foreground Service

Keeps Zloop running as a persistent overlay service. A system notification is always shown while Zloop is active.

Vibrate

Provides short haptic feedback when tapping buttons inside the app. No data involved.

6. Accessibility Service Disclosure

Zloop uses Android's Accessibility Service to observe which application is in the foreground. This is used solely to enable contextual reactions and personality responses. Zloop does not use this service to:

  • Read or capture your messages, emails, or notifications content
  • Access passwords, banking details, or sensitive form fields
  • Capture screenshots or record your screen

The Accessibility Service runs only while Zloop's overlay is active and is stopped when you deactivate Zloop.

7. Microphone & Voice Data Disclosure

Zloop's microphone access is used only to detect the wake word "Hey Zloop" and respond to subsequent voice commands. The App uses Android's built-in SpeechRecognizer API, which may route audio through Google's cloud speech services as part of the Android operating system — this behaviour is outside MammothDevs' control. MammothDevs does not receive, process, or store any voice data.

You can disable Zloop's voice features at any time from within the app settings.
